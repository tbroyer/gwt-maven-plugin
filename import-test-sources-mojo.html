gwt:import-test-sources

Full name:

net.ltgt.gwt.maven:gwt-maven-plugin:1.0-rc-3:import-test-sources

Description:

Adds all test-source directories as test resources.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-test-resources.

Parameters

(no parameters)
